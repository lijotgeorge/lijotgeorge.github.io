The Start

2021-08-26

I had dabbled in HTML/CSS many years ago but hadn’t really given it much serious thought or practice. Over the pandemic I kept thinking of going back and giving it another go, maybe not necessarily as a job prospect but as a back-up of a back up… just in case.

To that end I started searching through the internet (shoutout to r/learnprogramming) for the best places for a beginner to start and came up with the two most popular free web development courses – The Odin Project (TOP) and FreeCodeCamp (FCC). After reading a bit more about both I decided to start with TOP which turns out was just as well because they refer you to FCC’s course on web design anyway ðŸ¤·.

So, after going through the first set of courses on HTML and CSS, FCC had a whole list of projects you need to finish while TOP had just one. I decided to attempt two of the projects FCC suggested as well as the one by TOP.

FCC-1: Tribute Page

This is a fairly straightforward HTML page that teaches you the basics of setting up the various blocks in a page, including images in your page alongwith a caption if needed. I didn’t have much of a trouble with this one and managed to get it done in a few hours. The codepen for this page is attached below.

FCC-2: Survey Form

Another relatively straightforward HTML page this time focusing on creating a web form.

TOP-1: Google Homepage

This one was a bit tricky. The project was to recreate the Google homepage. I had some trouble getting flexbox to work the way I wanted it to (“Hardy Har Har!”, said every programmer ever) but eventually managed to get it done thanks the help of a couple of people from the TOP Discord server. I haven’t finetuned it as much as I would have liked but I still feel it’s a good first attempt.

This project also gave me a chance to upload my work on github (here) and host it as a separate URL (here).
